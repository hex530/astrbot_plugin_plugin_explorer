# AstrBot 插件探险家 (Plugin Explorer)

本插件为 AstrBot 提供了一个智能的插件扩展与管理方案。通过集成 GitHub 搜索与 Git 克隆功能，让你的 AI 具备自主发现并安装新技能的能力。

## 核心特性

- **智能搜索**：利用 GitHub API 搜寻带有 `astrbot-plugin` 标签的插件仓库。
- **Token 加速**：支持配置 GitHub 个人访问令牌（Token），突破 API 调用频率限制，搜索更稳定。
- **文档预览**：在安装前自动抓取 README 摘要，并支持自定义预览字数，帮助你快速判断插件功能。
- **一键安装**：支持通过指令直接克隆 GitHub 仓库到本地插件目录。
- **权限管理**：内置管理员校验，确保只有授权用户能进行安装操作。

## 安装方法

1. 在 AstrBot 插件管理中搜索 `plugin_explorer` 或通过本仓库 URL 安装。
2. 重启 AstrBot。

## 使用场景

- **探索**：对机器人说“搜搜看有没有 B 站相关的插件”。
- **进化**：确认文档符合需求后，下达安装指令，实现功能扩展。

## 配置项说明

| 配置项 | 说明 | 默认值 |
| :--- | :--- | :--- |
| `admin_qq` | 拥有搜索与安装权限的管理员 QQ。 | `1591793025` |
| `github_token` | GitHub 个人访问令牌，用于提升 API 限额。 | `(留空)` |
| `readme_summary_limit` | README 预览摘要的字数限制。 | `200` |
